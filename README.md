# ORPG

A text-based RPG built with vanilla JavaScript, Vite, and Tailwind CSS v4.

## Getting Started

```bash
npm install
npm run dev
```

## Commands

| Command        | Description                  |
| -------------- | ---------------------------- |
| `look` / `l`   | Examine your surroundings    |
| `go <dir>`     | Move (north/south/east/west) |
| `take <item>`  | Pick up an item              |
| `inventory`/`i`| Check what you're carrying   |
| `status`       | View your stats              |
| `help`         | Show available commands      |
